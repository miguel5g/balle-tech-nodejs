## Desafio 2 e 3

| Rota       | Método | Descrição                |
| ---------- | ------ | ------------------------ |
| /users     | GET    | Lista todos os usuários  |
| /users/:id | GET    | Busca um usuário pelo ID |
| /users     | POST   | Cria um novo usuário     |
| /users/:id | PUT    | Atualiza um usuário      |
| /users/:id | DELETE | Deleta um usuário        |
